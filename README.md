## Technology

- React-Typescript.
- TailwindCSS.
- Swiper (slider).

## Features

- Movie information (name, overview, casts,...).
- Watching free movies or trailer online.
- Suggested movies (popular, top rated, trending).
- Search by name.
- Filter by category.
